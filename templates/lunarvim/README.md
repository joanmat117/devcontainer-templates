# LunarVim DevContainer Template

Este template configura un entorno de desarrollo con LunarVim en Alpine Linux.

## ✨ Características

- Alpine Linux (ligero y rápido)
- LunarVim con Neovim 0.9.5
- Rust, Python, Node.js preinstalados
- Extensiones VS Code comunes
- Usuario no-root para seguridad

## 🚀 Cómo usar

1. En tu proyecto, crea la carpeta `.devcontainer`
2. Copia estos archivos
3. Abre en Codespaces o VS Code

## 📦 Incluye

- LunarVim (comando: `lvim`)
- Neovim vanilla (comando: `nvim`)
- Rust + Cargo
- Python3
- Node.js + npm + yarn
- Git, ripgrep, fd
